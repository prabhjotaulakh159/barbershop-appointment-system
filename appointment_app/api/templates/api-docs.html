POST http://localhost:5009/api/appointments

This endpoint creates an appointment with all the parameters and returns a status of 200

Any missing field or non-existant ID will return a status of 400 with some message

                
    curl -X POST \
    -H "Content-Type: application/json" \
    -d '{
            "status": "ON GOING",
            "date": "2024-01-01",
            "slot": "10-11",
            "venue": "Venue A",
            "client_id": 1,
            "professional_id": 2,
            "service_id": 1
        }' \
    http://localhost:5009/api/appointments
