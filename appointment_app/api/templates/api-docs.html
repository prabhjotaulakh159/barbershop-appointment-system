POST http://localhost:5009/api/appointments

This endpoint creates an appointment with all the parameters and returns a status of 200

Any missing field or non-existant ID will return a status of 400 with some message

                
    curl -X POST \
    -H "Content-Type: application/json" \
    -d '{
            "status": "ON GOING",
            "date": "2024-01-01",
            "slot": "10-11",
            "venue": "Venue A",
            "client_id": 1,
            "professional_id": 2,
            "service_id": 1
        }' \
    http://localhost:5009/api/appointments

GET http://localhost:5009/api/appointments/{id}

This endpoint gets an individual appointment by ID and returns a status 200

If the appointment ID does not exists, returns a status of 400 and some error message

                
    curl -X GET http://localhost:5009/api/appointments/1
    {
        'appointment_id': 1,
        'status': 'ON GOING',
        'date': '2024-01-01',
        'slot': '10-11',
        'venue': 'Venue A',
        'client_id': 1,
        'professional_id': 2,
        'service_id': 1
    }

DELETE http://localhost:5009/api/appointments/{id}

This endpoint deletes an individual appointment by ID and returns a status 200

If the appointment ID does not exists, returns a status of 400 and some error message

                
    curl -X DELETE http://localhost:5009/api/appointments/1

PATCH http://localhost:5009/api/appointments/{id}

This endpoint updates an appointment with a given ID with all the parameters and returns a status of 200

Any missing field or non-existant ID will return a status of 400 with some message

Here, we change the slot to 11-12 and the service ID to 4 for appointment ID 1

                
    curl -X PATCH \
    -H "Content-Type: application/json" \
    -d '{
            "status": "ON GOING",
            "date": "2024-01-01",
            "slot": "11-12",
            "venue": "Venue B",
            "client_id": 1,
            "professional_id": 2,
            "service_id": 4
        }' \
    http://localhost:5009/api/appointments/1
